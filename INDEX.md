## ⚡ Getting Started

### Step 1: Setup Backend
```bash
cd backend
npm install
npm run dev
```

### Step 2: Setup Frontend
```bash
cd frontend
npm install --legacy-peer-deps
npm run dev
```

### Step 3: Open Browser
```
http://localhost:5173
```

### Step 4: Create Account
1. Click "Sign Up"
2. Fill in details
3. Login with credentials

### Step 5: Browse & Book
1. Go to Vehicles page
2. Click on a vehicle
3. Enter dates
4. Click "Book Now"
